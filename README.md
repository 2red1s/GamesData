# GamesData

🎮 **Steam Games ID Database (RedSoft)**

---

### 🇷🇺 Русский
Этот проект содержит актуальный список ID всех игр Steam, доступных на платформе.
*   **Файл:** `redsoft_games.txt`
*   **Обновление:** Каждые 24 часа ⏱️
*   **Формат:** ID разделены запятыми для удобного парсинга.

---

### 🇺🇸 English
This project contains an up-to-date list of IDs for all Steam games available on the platform.
*   **File:** `redsoft_games.txt`
*   **Update Frequency:** Every 24 hours ⏱️
*   **Format:** IDs are comma-separated for easy parsing.
