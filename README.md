## > Как использовать / How to use / Қолдану жолы / 如何使用

### 🇷🇺 Инструкция
Чтобы у вас всегда были самые актуальные данные, используйте прямую ссылку на файл:
1. Нажмите на файл `redsoft_games.txt`.
2. Нажмите кнопку **"Raw"** в правом верхнем углу.
3. Скопируйте URL из адресной строки браузера.
4. Используйте эту ссылку в своем проекте для автоматического получения обновлений.

### 🇺🇸 Instructions
To ensure you always have the latest data, use the direct link:
1. Click on the `redsoft_games.txt` file.
2. Click the **"Raw"** button in the top right corner.
3. Copy the URL from your browser's address bar.
4. Use this link in your project to fetch updates automatically.

Әрқашан өзекті деректерді алу үшін тікелей сілтемені пайдаланыңыз:
1. `redsoft_games.txt` файлына басыңыз.

3. Браузердің мекенжай жолындағы сілтемені көшіріп алыңыз.
4. Жаңартуларды автоматты түрде алу үшін осы сілтемені жобаңызда қолданыңыз.
